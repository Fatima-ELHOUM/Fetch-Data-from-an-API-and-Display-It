# Task 2: Fetch Data from an API and Display It 🚀

The objective of this task is to retrieve data from a public API and present it in a user-friendly format.

## 📌 Project Overview
For this task, I built a **"Retro Joke App"**. It is a simple, interactive web application that fetches random jokes from a public API and displays the setup and punchline to the user with a clean, glassmorphism-inspired UI.

## ✨ Features
* **API Integration:** Fetches random jokes using the `Official Joke API`.
* **Asynchronous JavaScript:** Uses `async/await` and the `fetch()` API to handle network requests smoothly.
* **Modern UI/UX:** * Glassmorphism card effect using CSS `backdrop-filter`.
  * Loading animation (spinner) while waiting for the API response.
  * A delayed reveal for the joke's punchline to create a comedic timing effect.
* **Responsive Design:** Adapts perfectly to the screen size.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties (variables), animations, flexbox, and glassmorphism styling.
* **JavaScript (ES6+):** DOM manipulation, Event Listeners, and Fetch API.

## 🚀 How to Run
1. Clone this repository or download the source code files.
2. Ensure all files (`index.html`, `style.css`, `script.js`, and the background image) are in the same folder.
3. Open the `index.html` file in any modern web browser.
4. Click the **"Tell Me A Joke!"** button to fetch and display a new joke.
